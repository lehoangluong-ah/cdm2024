# SDK Changelog

:::note VERSIONING

The Chat SDK follows [Semantic Versioning](https://semver.org/). The version number is broken down into MAJOR.MINOR.PATCH.

:::

You can install a specific version of the Chat SDK by running:

```batch
npm install ahsdk
```
